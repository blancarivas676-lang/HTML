Proyecto: Sitio Web de Restaurante
Materia: Lenguajes de Marcado y Estilo Web y Gestores de Contenido

Descripción:
Este proyecto consiste en el desarrollo de un sitio web para un restaurante utilizando HTML, CSS, Bootstrap y JavaScript.

El sitio incluye las siguientes secciones:

* Inicio
* Sobre nosotros
* Menú
* Equipo
* Contacto

Este proyecto fue desarrollado con fines académicos como parte de una tarea universitaria.
